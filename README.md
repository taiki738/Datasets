# 画像データセット ラベリングプロジェクト

## 概要
本プロジェクトは、高品質な画像データセットを構築し、それを用いてVAE (Variational Autoencoder) およびLDM (Latent Diffusion Model) の訓練を行うことを目的としています。特に、UTKFaceデータセットをベースに、人種や年齢でフィルタリングされた顔画像に対して、主観的な「アリ」/「ナシ」といった評価を行うラベリングツールを開発・運用します。

## プロジェクトの目標
- カスタムデータセットで訓練されたVAEモデルおよびLDMモデルの作成。
- 表示される人の顔の画像に対して「アリ」か「ナシ」かといった主観でラベル付けをするためのラベリングツールの開発。
- 完成したツールを用いたアンケート結果の集計と、プロジェクトに最適なデータセットの構築。

## 現在の進捗
1.  **データセットの初期フィルタリング**:
    - UTKFaceデータセットから、特定の人口統計学的基準（年齢18〜25歳、人種：アジア人）に基づき画像をフィルタリングしました。
    - フィルタリングされた画像は `UTK-FACE/filtered_dataset/` ディレクトリに性別ごとに整理されています。

2.  **既存ラベリングツールの開発**:
    - Python (Flask) とHTML/CSS/JSを用いたWebアプリケーション形式のラベリングツールが `/image_labeler` ディレクトリに開発されました。
    - このツールは、モバイルデバイスからもアクセス可能であり、ユーザーは画像を「Keep」または「Discard」として分類し、その結果はそれぞれ `kept` または `discarded` サブディレクトリに保存されます。

## 新しいラベリングツールの方向性（アンケート形式）
- 複数人が参加し、匿名で画像を評価できるアンケート形式のラベリングツールへ移行します。
- 回答の重複を避けるための仕組みを導入しつつ、参加者の負担を最小限に抑える設計を目指します。
- 各画像に対する「アリ」/「ナシ」の二軸評価を基本とします。
- 参加者の任意で、年齢や性別などの最小限の属性情報を収集する機能を検討します。
- 評価結果はデータベースに保存し、後続のモデル訓練に活用しやすい形式で管理します。
